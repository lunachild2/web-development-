* 참조변수 사용하기

* 인스턴스와 힙 메모리
  - 힙 메모리- 객체 전용 메모리

* 참조 변수와 참조 값

  - 객체(힙메모리)의 주소를 가지고 있는 변수

---
# 자료형
* 기본자료형
	- 숫자를 저장하는 자료형(일정 공간 할당)
	- byte, short, int, long
	- float, double
	- char
	- boolean
	
* 참조자료형
	- 모든 클래스형 자료형
	- 객체 주소값
	- (4byte)
	
* 생성자
  - 객체를 생성하는 메서드
  - 반환값 - 생성된 객체의 주소값
  - 반환값은 변경 불가, 정의 X, return 키워드


* 함수의 시그니쳐
  - 패키지명 + 클래스명 + 자료형 + 함수명 + 매개변수 정의

  - 동일한 함수 시그니쳐를 정의 X -> 함수의 중복정의 -> 오류 발생

  - 함수의 시그니쳐가 다르면 여러개 정의할수 있는 것 -> 메서드 오버로드
