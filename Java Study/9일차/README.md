**1. 상속**
* 상속이란?

* 클래스의 상속

* 클래스 상속 문법
```
class 자식클래스 extends 부모클래스 {

}

C -> B -> A


C() -> super() : B() -> super() : A()


A객체 -> B 객체 -> C 객체
```

* super 
  - 지역변수 : 상위 클래스의 객체의 주소값
  - 생성자 메서드 : 상위 클래스의 생성자 메서드
    + super()

---

* 컴파일러가 추가 기본 추가
  - 기본 생성자
  - 모든 생성자 메서드의 첫번째 라인
    + super()
